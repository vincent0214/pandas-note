## 1 按行求和

# pandas.DataFrame.sum

```python
import pandas as pd

student = pd.read_excel("./student.xlsx",sheet_name="Sheet2")
row_sum = student[["语文","数学","英语"]].sum(axis=1) # axis=1按行计算.
print(type(student[["语文","数学","英语"]])) # 输出: Dataframe
print(row_sum) 
```

##  2 按行求平均值
